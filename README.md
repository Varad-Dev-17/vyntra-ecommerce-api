# ✨ Vyntra

&gt; _"Style that speaks, tech that delivers."_

💜 Vyntra is a modern fullstack e-commerce platform built for seamless shopping experiences — from browsing trendy collections to secure checkout.

---

## 🚀 Tech Stack

| Layer    | Tech                                          |
| -------- | --------------------------------------------- |
| Frontend | ⚛️ React + Vite + Tailwind CSS                |
| Backend  | 🟢 Node.js + Express                          |
| Database | 🍃 MongoDB Atlas                              |
| Media    | ☁️ Cloudinary                                 |
| Auth     | 🔐 JWT + OTP Email Verification (SMTP)        |
| UX       | 🔔 Toast Alerts + 🎬 Framer Motion Animations |

---

## 🎯 Purpose

A production-ready fashion e-commerce app with admin dashboard, real-time order management, and a polished glassmorphism UI.
